memoro
======

"A memorandum (memo) was from the Latin verbal phrase _memorandum est_, the gerundive form of the verb memoro, 'to mention, call to mind, recount, relate'" 
-- http://en.wikipedia.org/wiki/Memorandum

memoro is a WSGI app meant to be a lightweight personal globally-accessible "auxilary brain".  Mostly for entry of new concepts that you have not yet memorized, know that you'll need again in the future, and don't trust that you'll be able to recreate/relocate easily.
